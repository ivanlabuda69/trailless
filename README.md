> **trailless** — это чат в тёмном стиле, который не собирает ваши данные. Никакой телеметрии, никаких логов. Просто общение.

---

## ✨ особенности (которые ищут люди)

- 🔒 **Никакой слежки** — без метрик, без аналитики, без отправки данных третьим лицам
- 🌙 **Тёмная тема** — удобно для глаз в любое время суток
- 📧 **Регистрация** — по email или через Google (быстрый вход)
- 👤 **Личные чаты** — общайтесь по username, не зная номера телефона
- 👥 **Группы** — создавайте обсуждения с друзьями или командой
- ⭐ **Избранное** — сохраняйте важные диалоги сверху
- ✏️ **Редактирование и удаление** — исправляйте ошибки в сообщениях
- 💬 **Ответы на сообщения** — стройте понятные ветки обсуждения
- 🎁 **Подарки** — отправляйте виртуальные презенты друзьям
- ✅ **Верификация** — отмечайте официальные аккаунты галочкой
