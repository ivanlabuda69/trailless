## ✨ О проекте

**Trailless Messenger** — это легковесный, безопасный и интуитивно понятный мессенджер прямо в браузере. Он создан для тех, кто ценит своё время и конфиденциальность. Никаких лишних данных, только ваши сообщения.

🌐 **Попробуйте прямо сейчас:** [trailless.netlify.app](https://trailless.netlify.app)

---

## 🚀 Возможности

- 💬 **Мгновенные сообщения** — быстрая отправка и получение текста.
- 🔒 **Безопасность** — данные не хранятся дольше необходимого.
- 🌙 **Адаптивный дизайн** — удобно на ПК, планшете и телефоне.
- 🎨 **Минимализм** — интерфейс, который не отвлекает от главного.
- ⚡ **Высокая производительность** — моментальная загрузка и отклик.

---

## 🛠️ Технологии

- **Frontend:** React / Vue (или просто JS — уточните под ваш проект)
- **State Management:** (Redux / Pinia / Context API)
- **Styling:** CSS Modules / TailwindCSS
- **Hosting:** Netlify

```bash

# Перейдите в папку проекта
cd trailless-messenger

# Установите зависимости
npm install

# Запустите в режиме разработки
