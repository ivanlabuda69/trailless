## ✨ особенности

| | |
|---|---|
| `🧑‍💻` **Авторизация** | Вход по email + Google |
| `💬` **Личные чаты** | Общение по username |
| `👥` **Группы** | Создавай и приглашай |
| `✏️` **Редактирование** | ПКМ по своему сообщению → изменить |
| `🗑️` **Удаление** | ПКМ → удалить (только для себя) |
| `↩️` **Ответы** | ПКМ → ответить на сообщение |
| `🎁` **Подарки** | Система монет и подарков |
| `🖱️` **Контекстное меню** | Ответить / Редактировать / Удалить |
| `✅` **Верификация** | Белая галочка в чёрном кружке |
| `📱` **Мобильные** | Только ПК (заглушка с дизайном) |
